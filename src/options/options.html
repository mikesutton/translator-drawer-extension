How to get a Google Translate API Key

  1. Go to the Google Cloud Console
  2. Create a new project or select an existing one
  3. Enable the "Cloud Translation API"
  4. Create an API key from the Credentials page
  5. Copy the API key and paste it here

Note: Google Cloud Platform offers a free tier, but you may need to add billing information. Check Google's current pricing policy for details.
